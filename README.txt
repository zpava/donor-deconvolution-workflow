# donor-deconvolution-workflow

This workflow describes the steps to follow to identify subjects in a mixed sample used for 10x scRNAseq analysis based on unique SNPs present.
The reference used genome1K.phase3.SNP_AF5e2.chr1toX.hg38.vcf.gz downloaded from https://sourceforge.net/projects/cellsnp/files/SNPlist/ 
The workflow uses the following software:
python3
samtools v1.9
cellsnp-lite: https://cellsnp-lite.readthedocs.io/en/latest/ 
vireo: https://vireosnp.readthedocs.io/en/latest/
